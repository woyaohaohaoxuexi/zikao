#### 求资源精确替代率
```
已知生产函数  y = 18x1 - x1^2 + 14x2 - x2^2 当x1 = 5, x2 = 4 时，计算 x2 替代 x1 的精确边际替代率?
  解： MRS = - MPPx2 / MPPX1 = 14 - 2x2 / 18 - 2x1  = - 6/8 = -0.75 （一定要注意：是负的）
```
#### 求边际产量
```
某农场棉花生产函数模型 y = 5x1^1/2 * x2^1/3 * x3^1/2 ，x1、x2、x3表示氮、磷、钾肥投入量，y 表示棉花产出量，若x1、x2、x3平均施用量分别为9、8、25，由模型计算得到棉花产量y＝150公斤，则氮肥（x1）的边际产量是?
解：(1/2) * (150/9) = 25/3 
```
